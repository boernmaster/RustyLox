Loxone UDP Push

Sends weather values to the Miniserver via UDP virtual inputs, identical to the weather4lox protocol.

Loxone Cloud Emulator

⚠️ Miniserver Gen 1 only. The cloud emulator uses the legacy weather.loxone.com:6066 weather protocol which is only supported by Loxone Miniserver Generation 1. Gen 2 and newer Miniservers use a different weather API and are not compatible with this feature.

Serves the weather.loxone.com:6066 REST API format so the Miniserver can poll RustyLox instead of the real Loxone cloud. Optionally configure dnsmasq to redirect the DNS name automatically.

Requires dnsmasq installed and write permission to /etc/dnsmasq.d/.
IP address as seen by the Miniserver. Used in dnsmasq config and UDP push target.
